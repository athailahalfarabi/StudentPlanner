
## Usage Mobile
1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

## Path
Mobile di folder student-planner
Backend di folder backend

# Commit Convention
Biasa gunakan commit yang jelas seperti [ADD], [Refactor] 
pisahkan commit backend dan mobile [BACKEND]/[MOBILE]
